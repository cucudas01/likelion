# Mission 04
작성자: 양찬승

## 과제 설명
데이터 관리 도구 변경: 기존의 고정된 크기인 배열(Array) 대신, 크기가 자유롭게 변하는 ArrayList와 키-값 쌍으로 저장하는 Map을 사용합니다.

제네릭(Generics) 적용: List<Lion>처럼 타입을 명확히 지정하여 프로그래밍의 안정성을 높입니다.

데이터 무결성 확보: 새로운 멤버를 추가할 때 이미 존재하는지 확인하는 중복 체크 로직을 구현합니다.

조회 기능 강화: 저장된 데이터에서 특정 이름을 검색하거나, 특정 파트(백엔드, 디자인 등) 멤버만 모아보는 필터링 기능을 만듭니다.
